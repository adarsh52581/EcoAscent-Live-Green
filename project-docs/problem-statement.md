# Problem Statement

Most carbon trackers show numbers. Numbers don't change behavior — **feeling** does.
EcoAscent reframes the user's daily choices as a **Living World** that visibly thrives,
strains, or chokes in response to the actions they log.

## Target user
Climate-curious people who already know "flying is bad" but haven't internalized scale.

## Core objectives
1. Emotional feedback loop within < 1 second of logging an action.
2. Zero onboarding friction — works offline, no sign-up, no API keys.
3. Gamified social comparison via a neighborhood leaderboard.

## Non-goals
- Scientific-grade CO₂ accounting.
- Heavy 3D visualizations that slow phones down.
